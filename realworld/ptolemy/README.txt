The models in this folder and its subfolders were imported and transformed
into ELK-specific file formats from files that are part of the Ptolemy II
project. The project can be found here:

    http://ptolemy.eecs.berkeley.edu/ptolemyII/

The files are located in the project's SVN repository:

    https://chess.eecs.berkeley.edu/ptexternal/

Specifically, they are in subdirectories of ptolemy/domains.
